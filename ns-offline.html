STEP 02 Setup Database (XAMPP)

  1. Install XAMPP, jalankan XAMPP Control Panel.
  2. Klik Start pada Apache dan MySQL.
  3. Buka browser, ketik localhost/phpmyadmin.
  4. Buat database baru nama: ninjasage.
  5. Import file ninjasage.sql dari starter pack ke dalam database tersebut.

STEP 03 Konfigurasi Laravel Server

  1. Ekstrak Folder Server ke D:\NinjaSagaOffline.
  2. Rename file .env.example jadi .env.
  3. Buka CMD di folder tersebut, lalu ketik perintah ini:
# 1. Menginstal sistem server 
composer install
# 2. Membuat kunci keamanan 
php artisan key:generate
# 3. Sinkronisasi data game 
php artisan migrate --seed

STEP 04 Patching SWF (Agar Connect)

Agar game tidak error saat loading, edit file SWF menggunakan JPEXS:

  • Buka JPEXS > Open public/game/NinjaSage.swf.
  • Folder scripts > Cari ZendAMFClient.as > Ganti link ke "http://127.0.0.1:8000/amf".
  • Cari StorageLoader.as > Ganti return link ke "http://127.0.0.1:8000/game_data/".
  • Save All.

🚀 CARA MENJALANKAN GAME

  1. Buka XAMPP (Start Apache & MySQL).
  2. Buka CMD di folder server, ketik: php artisan serve.
  3. Buka folder public/game, klik 2x pada NinjaSage.exe.
  4. Selesai! Selamat bermain, Ninja!
