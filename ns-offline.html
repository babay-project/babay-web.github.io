STEP 02 Setup Database

  1. Jalankan XAMPP, Start Apache & MySQL.
  2. Klik tombol di bawah untuk buka database:
🌐 Buka phpMyAdmin
  1. Buat database baru dengan nama: ninjasage.
  2. Import file ninjasage.sql dari starter pack.

STEP 03 Konfigurasi Server

Ekstrak server ke D:\NinjaSagaOffline. Buka CMD di folder tersebut:

# Langkah 1: Install Library (Jika error, gunakan "composer update") 
composer install
# Langkah 2: Generate Key 
php artisan key:generate
# Langkah 3: Sinkronisasi Database 
php artisan migrate --seed

STEP 04 Patching SWF

Gunakan JPEXS Decompiler untuk mengedit public/game/NinjaSage.swf:

  • ZendAMFClient.as: Ganti ke "http://127.0.0.1:8000/amf"
  • StorageLoader.as: Ganti return ke "http://127.0.0.1:8000/game_data/"

🚀 CARA MENJALANKAN GAME

  1. XAMPP wajib menyala (Apache & MySQL).
  2. Buka CMD di folder server, ketik: php artisan serve.
  3. Buka public/game/NinjaSage.exe.
